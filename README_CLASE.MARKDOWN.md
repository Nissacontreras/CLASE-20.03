# CLASE-20.03
Github es una plataforma en la nube diseñada para desarrolladores, que permite almacenar, gestionar y compartir código fuente utilizado el sistema de control de versiones Git. Funciona como una red social para programadores, facilitando la colaboración, el seguimiento de cambios.
Markdown es un lenguaje de mercado ligero. En terminos simples, es una forma de escribir un texto con formato añadiendo unas simples marcas de texto.


Si quieres separar dos parrafos, debes presionar enter dos veces. Esto deja un espacio en blanco entre los bloques de texto.
